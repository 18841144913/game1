# Monster Survivors 怪物幸存者

一个基于 HTML5 的多语言(中英双语)游戏网站。游戏为紧张刺激的生存动作类型,玩家需要在不断涌来的怪物浪潮中生存下去。

## 🎮 游戏特色

- 响应式网页设计,完美适配PC和移动端
- 中英文语言切换功能
- 动态昼夜循环系统
- 50+ 种独特怪物类型
- 渐进式武器升级系统  
- 多人生存模式
- 全球排行榜

## 🛠 技术栈

- HTML5
- Tailwind CSS
- JavaScript
- iframe 游戏嵌入

## 🚀 快速开始

1. 克隆项目到本地:
```bash
git clone https://github.com/your-username/monster-survivors.git
```

2. 使用浏览器打开 index.html 文件即可运行

## 🎯 游戏操作

- 使用 WASD 或方向键移动
- 使用鼠标瞄准和射击
- 收集道具提升战斗能力
- 在怪物浪潮中尽可能生存
